# Folklorist

Study / lore skills and herbal poison/heal style casts. **Minor protection** fading is highlighted. Several skills **require a target** or they print “No target!” in the client.

## Line highlights

**The minor protection fades away** in bold red.

## Shortcuts

### Skills

| Shortcut | Full name | Notes |
|----------|-----------|-------|
| `usc` | Study creature | Target required (`use study creature at …`). |
| `uel` | Eye of loraen | Target required. |
| `upl` | Plant lore | Target required. |

### Spells

| Shortcut | Full name | Notes |
|----------|-----------|-------|
| `cpb` | Poison blast | No args: `cast 'poison blast'`. With target: `target`; `cast poison blast at …` (unquoted targeted form). |
| `cvs` | Venom strike | Same pattern as poison blast (quoted vs unquoted branches). |
| `cts` | Thorn spray | No args: `cast 'thorn spray'`. With target: `cast 'thorn spray' <target>`. |
| `chb` | Herbal poison blast | Target required: `cast 'herbal poison blast' <t> use herb`. |
| `chh` | Herbal healing | No args: `cast 'herbal healing' me use herb`. With args: tail after spell name + `use herb`. |
| `cmp` | Minor protection | No args: `cast 'minor protection' me`. With args: `cast 'minor protection' <target>`. |
| `cfp` | Field of poison | `cast field of poison` (unquoted). |

!!! warning "Shared shortcut"
    **`chb`** is also **Seminary harm body** if that guild comes first — pick guild order carefully.

## Profile

Enable **`folklorist`** in **`/guilds`**.
