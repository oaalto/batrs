# Liberator

Ghost spells usually paired with **`liberator select`** on the right ghost role (non-specialist, armoursmith, weaponsmith, guardian, ranger, soldier). Highlights misses, light fading, and ghost farewell whispers.

## Line highlights

Red on swing miss and holy buff drop; ghost farewell whisper in red.

## Shortcuts

### Skills

| Shortcut | Full name | Notes |
|----------|-----------|-------|
| `us` | Slash | Standard `use` targeting. |
| `ugs` | Ghost slash | No args: `liberator select clear; liberator select weakest soldier; use 'ghost slash'`. With target: prefixes `target`. |
| `urs` | Radiant slash | **`liberator select`** non-specialist list + `use 'radiant slash'` (optional target prefix). |
| `ugcf` | Ghost channeling (fire) | `liberator select weakest ranger`; `use 'ghost channeling' fire`. |
| `ugcc` | Ghost channeling (camp) | Same with **`camp`** instead of fire. |

### Spells

| Shortcut | Full name | Notes |
|----------|-----------|-------|
| `cgl` | Ghost light | Non-specialist **`liberator select`**, then `cast 'ghost light'` (optional `target` prefix + spell target). |
| `cgc` | Ghost chill | Same pattern with **`ghost chill`**. |
| `cga` | Ghost armour | **`liberator select weakest armoursmith`**; `cast ghost armour`. |
| `cgs` | Ghost sword | **Target required**: **`liberator select weakest weaponsmith`**; `cast ghost sword at <target>`. |
| `cgcom` | Ghost companion | **`liberator select weakest guardian`**; `cast ghost companion`. |
| `clink` | Ghost link | Non-specialist select; `cast ghost link at me`. |
| `crs` | Restful sleep | Non-specialist select; `cast restful sleep` (optional trailing args). |
| `chg` | Holy glow | `cast holy glow` (unquoted). |

### Other

| Shortcut | Full name | Notes |
