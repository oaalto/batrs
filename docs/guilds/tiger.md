# Tiger

Tiger combat highlights, **sect** cultivation coloring (same as Monk), and **meditation** that dismounts first if a mount flag is set.

## Line highlights

Red/green for resists, flame fists ending, whiffs, stuns, kill leaps. Plus shared sect cultivation colors.

## Shortcuts

### Skills

| Shortcut | Full name | Notes |
|----------|-----------|-------|
| `ip` | Iron palm | Standard `use` targeting. |
| `dm` | Dim mak | Target required (`target …;use 'dim mak' at …`). |
| `med` | Meditation | If the Tzarakk mount-summoned flag is set, **`dismount`** is sent first. |
| `upl` | Pick locks | Target required. |
| `usn` | Sneak | Sends `use 'sneak'`. |

### Spells

| Shortcut | Full name | Notes |
|----------|-----------|-------|
| `tc` | Tiger claw | Standard `cast` targeting. |
| `cff` | Flame fists | Sends `cast 'flame fists'`. |

!!! note "Shared shortcuts"
    **`ip`** overlaps Monk iron palm. **`med`** overlaps Monk / Psionicist / Tzarakk.

## Profile

Enable **`tiger`** in **`/guilds`**. If you play with Tzarakk mount detection, guild order can matter for **`med`**.
